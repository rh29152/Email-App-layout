# Email App Layout  

A minimal **Email App UI** created with **HTML & CSS Grid**.  
The layout includes:  
- Top navigation bar  
- Action buttons (New, Reply, Forward, etc.)  
- Inbox section with sample emails  
- Main email view  
- Additional info sidebar
